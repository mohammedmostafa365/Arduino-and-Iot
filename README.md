## üí° How to Run

1. Open the `.ino` file for the project you want in the **Arduino IDE**.  
2. Connect your **Arduino board** via USB.  
3. Select the correct **board type** and **port** in *Tools ‚Üí Board/Port*.  
4. Upload the code and test your circuit.  
5. Observe the results or control via Bluetooth/IoT as per the project.
